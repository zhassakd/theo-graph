# 贡献指南

感谢你对 TheoGraph 的兴趣！

## 基本原则

TheoGraph 是一个以**概念节点**为核心的中文神学知识库。所有贡献应当遵循以下原则：

- **引用优于创造**：优先引用经典文献（圣经、教父著作、信经），而非提出全新的神学理论
- **平信徒可读**：目标读者是"对神学有兴趣但没有受过系统训练的普通人"
- **一个概念一个文件**：每个概念节点自包含，可以独立阅读
- **双向链接**：使用 `[[概念名称]]` 格式链接到其他节点

## 文件命名

- 概念：`概念名称.md`（如 `恩典.md`、`三位一体.md`）
- 系统论述：`数字_章标题.md`（如 `01_认识神.md`）
- 经典笔记：`卷数字_卷标题.md`（如 `卷01_婴孩与少年.md`）

## 概念节点格式

每个概念节点建议包含：

```markdown
# 拉丁术语 · 中文术语

> 出处：作者《著作名》中的核心命题

---

正文（平信徒可读的解释）

---

## 相关章节
- [[01_认识神]] — 简短注释

## 关联概念
- [[概念/恩典]] — 简短注释

## 关键人物
- 奥古斯丁
- 路德
```

## 双向链接约定

- 跨目录链接使用完整路径：`[[概念/恩典]]`、`[[system/01_认识神]]`
- 同一目录内链接使用文件名：`[[恩典]]`
- 链接后跟 ` — ` 和简短注释说明关系

## 提交规范

- 一个 Pull Request 应该只涉及一个概念或一个章节
- 提交信息（commit message）使用中文，说明做了什么改动
- 修改现有概念时，请说明修改理由

## 许可

所有贡献采用与项目相同的 **CC BY-SA 4.0** 协议。提交贡献即表示你同意此许可。
