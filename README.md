# TheoGraph · 中文神学概念网络

[![License: CC BY-SA 4.0](https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-sa/4.0/)

**一个以概念节点为核心、交叉引用为骨架的中文神学知识体系。**

不是论文集，不是教材 PDF，不是讲道集——而是一张可以浏览、检索、编辑、复用的**神学知识图谱**。

---

## 为什么做这个

中文互联网上的神学资源，大多是这样：

- PDF 扫描的经典著作 → 无法搜索、无法引用
- 教牧博客文章 → 碎片化、缺乏体系
- 神学院讲义 → 学者写给学者、门槛极高

TheoGraph 试图换一种方式：**用软件工程的方法组织神学知识。**

每一个神学概念（启示、恩典、三位一体、称义……）是一个独立的 Markdown 文件，彼此通过双向链接连接。你可以从任何一个概念出发，沿着链接走遍整张网络。

## 设计原则

### 每个概念节点是独立的

每个概念可以单独阅读，不需要前后翻页。但每个节点都标注了**前置概念**（读这个之前你需要先知道什么）和**关联概念**。

### Markdown 优先

纯文本，Git 友好，任何编辑器都能打开。用 Obsidian 或任何支持 `[[双向链接]]` 的工具打开效果更好。

### 平信徒可读

目标读者是"对神学有兴趣但没有受过系统训练的普通人"，不是神学院里的学者。每一篇的目标读者是"对神学有兴趣但没有受过系统训练的普通人"。

### AI 可检索

每个概念节点自包含、结构化，适合作为 AI Agent 的检索增强（RAG）知识库。配合 AI 可以获得对话式神学辅导体验。

## 使用方式

### 🧠 在 Obsidian 中打开

```bash
git clone https://github.com/zhangpengcheng/theo-graph.git
# 用 Obsidian 打开 theo-graph 文件夹作为 vault
```

### 🤖 作为 AI 知识库

将所有 `.md` 文件导入 CherryStudio / RAG 系统的知识库模块，AI 将能够用这些内容回答神学问题。

### 📖 在线阅读

每个 Markdown 文件在 GitHub 上可以直接阅读。概念之间的 `[[链接]]` 点击跳转。
